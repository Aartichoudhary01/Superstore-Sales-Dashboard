# Superstore-Sales-Dashboard
## Project Overview
The objective of this project is to analyze the sales and profit performance of a retail superstore using Power BI. The dashboard aims to provide meaningful insights into product categories, customer segments, regional performance, and time-based trends to support data-driven decision-making and improve overall business profitability.
## Dataset Used
The dataset used in this project is the Sample Superstore dataset, which contains transactional retail data.

Key columns include:
- Order Date
- Region (East, West, Central, South)
- City
- Category & Sub-Category
- Segment (Consumer, Corporate, Home Office)
- Sales
- Quantity
- Profit

This dataset represents sales operations across different regions and product categories in a retail business environment.
## Questions And KPIs
The dashboard is designed to answer the following key business questions:

- What is the total sales, profit, and quantity sold?
- How does profit vary over time (year and quarter)?
- Which product categories and sub-categories are most profitable?
- Which regions contribute the most to profit?
- How do different customer segments impact profitability?
- Which cities or regions are underperforming?
- What are the top and bottom performing sub-categories?
## Process
The project was developed following these steps:

1. Data Cleaning:
   - Checked for missing or inconsistent values
   - Standardized column names and formats

2. Data Transformation:
   - Created measures using DAX (Total Sales, Total Profit, Total Quantity)
   - Aggregated data for time-based analysis (Year, Quarter)

3. Data Modeling:
   - Ensured proper relationships between fields for accurate analysis

4. Dashboard Design:
   - Created a dark-themed, visually appealing layout
   - Organized visuals for intuitive navigation

5. Visualization:
   - Used KPI cards for key metrics
   - Line chart for time-based profit trends
   - Bar chart for sub-category performance
   - Donut charts for category, segment, and regional distribution
   - Table for detailed drill-down analysis
   - Slicers for interactive filtering (City)

## Project Insights
- Technology category contributes the highest share of total profit.
- Copiers and Accessories are the most profitable sub-categories.
- Some sub-categories like Tables and Bookcases show negative or low profit, indicating potential loss areas.
- The Consumer segment generates the highest profit compared to Corporate and Home Office.
- The West region leads in profitability, followed by the East region.
- Profit shows fluctuations across quarters, with noticeable peaks in certain periods.
## Final Conclusion
The Superstore Sales Dashboard provides a comprehensive view of business performance across multiple dimensions such as time, category, region, and customer segment.

The analysis highlights key profit-driving areas as well as underperforming segments, enabling stakeholders to make informed decisions. By leveraging these insights, businesses can optimize product offerings, improve operational efficiency, and enhance profitability.
